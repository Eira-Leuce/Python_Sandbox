# Python_Sandbox
Простые листинги с 1 курса
## 1. Chess Board
* **Стек:** `Python`
* **Концепции:** Вложенные циклы, Работа с 2D-координатами.
* **Описание:** Программа выводит в консоль последовательность символа '*' и чисел в виде шахматной доски.
## 2. Discont count
* **Стек:** `Python`
* **Концепции:** Условные операторы, Базовая математика.
* **Описание:** Программа рассчитывает размер скидки в зависимости от суммы покупки.
## 3. Encriptor
* **Стек:** `Python`
* **Концепции:** Побитовые операции
* **Описание:** Программа моделирует самый примитивыный метод шифрования при помощи операуии логического исклющего ИЛИ. 
## 4. Heart
* **Стек:** `Python`
* **Концепции:** Вложенные циклы.
* **Описание:** Программа выводит в консоль последовательность символа '*' в виде сердца.
## 5. Hourglass
* **Стек:** `Python`
* **Концепции:** Вложенные циклы, Обработка исключений.
* **Описание:** Программа выводит в консоль последовательность символа '*' и чисел в виде песочных часов.
## 6. Rectangle
* **Стек:** `Python`
* **Концепции:** ООП (Классы, методы).
* **Описание:** В программе создан класс Rectangle, который описывает объект - прямоугольник. Методы класса помогают вычислить площадь и периметр фигуры.
## 7. Year check
* **Стек:** `Python`
* **Концепции:**  Логические операторы.
* **Описание:** Программа запращивает у пользователя год и предсказывает, является ли он високосным или нет.
## 8. Bank account
* **Стек:** `Python`
* **Концепции:**  ООП.
* **Описание:** В программе создается класс BankAccount, позволяющий производить выполнять простейшие банковские операции.
## 9. Vector
* **Стек:** `Python`
* **Концепции:**  ООП, перегрузка операторов, методы доступа
* **Описание:** В программе создан класс Vector, сеттер и геттер, выполнена перегрузка операций сложения и вычитания, которые теперь работают с векторами.
